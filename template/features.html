MET3220 Project

Florida Climate Analysis

An overview of the features developed with this project.

Data Analysis
Data has been organized from nearly 100 weather stations throughout Florida, providing valuable insights into weather patterns and trends throughout the state.
Interactive maps
Interactive weather maps allow users to explore historic weather conditions, providing a more engaging and informative experience.
Map Markers
Custom map markers—colored based on trends—are a powerful tool for visualizing trends, patterns, and insights. Quick Trend graphs are avaiable in the marker.
Trend Plots
Trend plots of past weather data offer a visual representation of long-term weather patterns, allowing for insight into climate trends and changes over time.
Time Series Plots
Time series graphs of historic weather data every 30 years can provide valuable insights into how the climate has changed over time. These graphs can show trends in temperature, precipitation, and other weather variables that can help us understand how the climate is evolving.
Github Integrated
This entire project is avaiable within the Github repo MET3220, including the webframe design, scripts to create plots, data files, and weather stations.
